# If un string con cero

**Sí lo hará**.

Cualquier string excepto uno vacío (y "0" que no es vacío) se convierte en true en un contexto lógico.

Podemos ejecutar y verificar:

````js
if ("0") {
  alert( 'Hola' );
}
````
