## Mostrar una alerta con un script externo

El código HTML:

````html
<!DOCTYPE html>
<html>

<body>

  <script src="alert.js"></script>

</body>

</html>
````

Para el archivo alert.js en la misma carpeta:

````js
alert("¡Soy JavaScript!");
````
