## Una pagina simple

Código JavaScript:

````js
let name = prompt("¿Cuál es tu nombre?", "");
alert(name);
````

La página completa:

````html
<!DOCTYPE html>
<html>
<body>

  <script>
    'use strict';

    let name = prompt("¿Cuál es tu nombre?", "");
    alert(name);
  </script>

</body>
</html>
````
